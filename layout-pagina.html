 As tabelas permitem que o desenvolvedor web organize os dados em linha se colunas. Um conjunto de dados estruturados.
Exemplo: título <th>; linhas <tr>; colunas <td>.
Além disso, também podemos estilizar as tabelas, utilizando as propriedades informados no menu ao lado.

Resumo

Border permite a estilização e a alteração no formato das bordas;

Size permite manipular o tamanho de cada linha, coluna ou mesmo da tabela inteira;

Padding faz com que ocorra o preechimento nas células da tabela;

Spacing altera os espaços entre as células de uma tabela;

Colspan mescla as células de acordo com a quantidade informada de colunas;

Rowspan mescla as células de acordo com a quantidade informada de linhas;

Zebra Stripes adiciona uma cor ao fundo das linhas da tabela, de forma alternada, semelhante a pintura de uma zebra;

Hover destaca a célula em que o cursor está apontando;

Colgroup permite a estilização de uma coluna em sua tabela.
